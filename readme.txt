== Description ==
Cleanup is Directory Listing & Classifieds for Wordpress sites

[Demo](https://cleanup-demo.e-plugins.com/)  |  [Videos](https://www.youtube.com/watch?v=yP9Kp802qos)  |  [Docs](https://help.eplug-ins.com/cleanup/)  |  [Support](http://e-plugins.com/support)  |  [Features](https://e-plugins.com/cleanup-main/)

Introducing Cleanup - Directory Listing & Classifieds WordPress Plugin, the ultimate solution for creating powerful directory portals on WordPress sites. Fully responsive and compatible with any SEO plugin and WordPress theme, Cleanup revolutionizes the way you manage listings.

#### Key Features:

*Robust Listing Search: Allow users to easily search and find listings based on various criteria.

*CSV Listing Importer: Quickly import listings in bulk using CSV files for effortless management.

*Rating System: Enable visitors to rate and review listings, providing valuable feedback.

*PDF Maker: Generate PDF documents for listings, making it easy to share and print information.

*Multilingual Support: Fully compatible with WPML, Loco Translate, and other translation plugins for global reach.

*User Registration: Allow users to register on your site and create their own listings.

*Frontend Dashboard: Provide users with a frontend dashboard to manage their listings and account settings.

*User Settings: Customize user settings to meet their preferences and needs.

*User Profile Management: Empower users to edit their profiles and manage their directory listings effortlessly.

*Listing Notifications: Keep users informed with notifications for new listings, updates, and more.

*Saved Listings: Allow users to save their favorite listings for quick access.

*User Photo Upload: Enable users to upload photos to accompany their listings, enhancing visual appeal.

*Email Templates: Customize email templates for various notifications, including welcome emails, password resets, and listing notifications.

*User Directory: Create a comprehensive user directory with profile pages for each user, promoting community engagement.

*And Much More: Explore a wide range of additional settings and features to tailor the directory to your specific needs.

With Cleanup, you can build dynamic, user-friendly directory websites that cater to your audience's needs. Get started today and unlock the full potential of your directory platform. Visit our [Demo](https://cleanup.e-plugins.com/) to experience Cleanup in action, or explore our [Features](https://e-plugins.com/cleanup-main/) page for more information.
